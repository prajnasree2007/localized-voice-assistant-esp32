# Localized Voice Assistant for Senior Citizens

## Overview
The Localized Voice Assistant for Senior Citizens is a privacy-focused offline assistive system developed using ESP32. The system is designed to help elderly people through voice interaction, medicine reminders, emergency handling, and smart home assistance.

The assistant supports localized interaction and operates without requiring continuous internet connectivity, making it more secure, reliable, and user-friendly for senior citizens.

## Working Principle
The ESP32 acts as the main controller of the system and processes offline voice interactions.

The assistant provides medicine reminders and appointment alerts using RTC scheduling. Emergency situations can trigger GSM-based alert handling.

The system also supports smart home appliance control using relay modules. A servo-based medicine box helps organize medication access for senior citizens.

Voice responses and interactions are provided through the audio output system while important information is displayed on the OLED screen.
